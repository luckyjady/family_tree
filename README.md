<b>家谱系统，逐渐完善中... </b><br>

家谱是一脉相承，一脉相传的东西，我们的祖宗是谁？有哪些丰功伟绩，我们的家训是什么？翻开家谱，我们可以回忆什么？又给我们或者后人哪些启示呢？<br>

背景：
本人是农民三代，家住农村，老家里有一本97年撰写的族谱，某天在想，现在是数字时代，传统的家谱书，修编成本很高，高效传阅和快速搜索都是问题，何不做一个家谱系统呢？
